# DBMS Project

DBMS is a Bash script DataBase Management System

# installation & Run
Use the version control tool [git](https://git-scm.com/) to clone the project.

    git clone https://github.com/AmrTarek17/DBMS-Project.git
    cd DBMS-Project
    ./run.sh

Or press "<span style="color:green">Code</span>" button then chose "Download ZIP"

# Our Features

## DataBase layer

- Create Database
- List DataBase
- Connect To DataBase
- Drop DataBase

## Table Layer

- Crate Table
- List Tables
- Drop Table
- Select Table

## Record Layer

- Insert Into Table
- Select From Table
- Delete From Table
- Update Table
